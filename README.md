## 📋 **PRASYARAT**

Sebelum menggunakan tools ini, kamu WAJIB punya:

1. ✅ **Akun Meta Developer** (gratis)
2. ✅ **WhatsApp Business Account** (gratis)
3. ✅ **Phone Number ID** (dari dashboard)
4. ✅ **Access Token** (dari dashboard)
5. ✅ **Nomor telepon tujuan** (yang mau dikirimi pesan)

> ⚠️ **PENTING:** Token dari dashboard hanya berlaku **24 jam**. Untuk permanent, buat **System User** di Business Manager.
